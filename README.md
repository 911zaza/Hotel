# Hotel
gestion d'hotelrie

## Résumé du projet
Ce dépôt contient une application de gestion hôtelière composée d'un frontend React (dossier `hotel-ui`) et d'une API backend Python (FastAPI).

Principales choses réalisées :

- Frontend : logo remplacé, header dynamique (transparent → flou au scroll), sections hero avec images d'arrière-plan, upload & preview d'images pour événements/plats, composant footer, styles globaux et utilitaires CSS.
- Backend : ajout des champs `evenement_url` et `plat_url` dans les modèles, endpoints d'upload d'images, utilitaires de sauvegarde d'images, et exposition du dossier `images/` en statique.

Consulte [PRESENTATION.md](PRESENTATION.md) pour un résumé plus détaillé.
